# Java lab 2.

## Вариант 1:
Задача: **Префиксное дерево (Trie)** для работы со строками (малыми латинскими буквами `a`–`z`).

Консольное меню: добавить слово, проверить наличие слова, проверить наличие слов по префиксу, получить список слов по префиксу, посчитать количество слов с префиксом, удалить слово, показать все слова, загрузить/сохранить слова из файла, сохранить визуализацию дерева в формате DOT.

---

## Файлы

- `src/Trie.java` — реализация префиксного дерева:
  - хранение слов в виде префиксного дерева;
  - вставка, поиск и удаление слов;
  - поиск по префиксу;
  - подсчёт количества слов с заданным префиксом;
  - генерация графа в формате Graphviz DOT;
  - вывод всех слов в виде строк.

- `src/TrieConsoleApp.java` — консольное приложение:
  - интерактивное меню;
  - работа с пользовательским вводом;
  - загрузка/сохранение слов в файл (по одному слову в строке);
  - сохранение DOT-представления дерева в файл;
  - обработка ошибок ввода и ошибок при работе с файлами.

---

## Ограничения

- Используются **только массивы**, без готовых коллекций Java (`List`, `Set`, `Map` и т.д.).
- Trie работает со строками, содержащими только символы `a`–`z`.  
  При вводе других символов выбрасывается исключение, а пользователю выводится сообщение об ошибке.
- Дубликаты слов не добавляются.

---

## Дополнительные функции

По сравнению с базовым заданием реализовано:

- `int size()` — количество уникальных слов в дереве.
- `boolean remove(String word)` — удаление слова из дерева с очисткой лишних узлов.
- `int countByPrefix(String prefix)` — количество слов, начинающихся с заданного префикса.
- `String toDot()` — генерация строкового представления дерева в формате **Graphviz DOT** для визуализации структуры.

Также:

- Есть методы загрузки и сохранения слов через консольное приложение (работа с файлами и обработкой исключений).
- Все пользовательские сообщения в консоли — на русском языке.

---

## Консольное меню

Приложение `TrieConsoleApp` предоставляет следующие команды:

1. **Добавить слово**  
2. **Проверить, есть ли слово**  
3. **Проверить, есть ли слова с префиксом**  
4. **Вывести слова по префиксу**  
5. **Подсчитать количество слов по префиксу**  
6. **Показать размер (количество слов)**  
7. **Удалить слово**  
8. **Показать все слова в дереве**  
9. **Загрузить слова из файла** (по одному слову в строке)  
10. **Сохранить все слова в файл**  
11. **Сохранить DOT-представление в файл**  
0. **Выход**

## Как запустить

1. Открыть проект в IntelliJ IDEA.
2. Убедиться, что в папке `src` лежат файлы:
   - `Trie.java`
   - `TrieConsoleApp.java`
3. Открыть файл `TrieConsoleApp.java`.
4. Запустить `main`-метод:
   - правый клик по файлу → **Run 'TrieConsoleApp.main()'**.

Внизу откроется консоль с меню. Управление осуществляется вводом номера команды и нажатием Enter.
